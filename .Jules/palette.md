## 2025-03-26 - Explicit Range Slider Values for Predictability
**Learning:** Users often struggle with arbitrary sliders that lack explicit numerical feedback, as they do not provide clear affordances of exact changes or allow the user to return to a previous exact state. Providing `<span className="settings-panel__hint">` alongside every `<input type="range">` ensures a predictable and stable UX across settings. It also requires rigorous type checking and fallback logic (like `.toFixed(1)` via nullish coalescing `??`) when formatting arbitrary store values to prevent UI crashes.
**Action:** Always verify that input sliders include real-time text feedback adjacent to them, paired with rigorous data sanitization (casting to `Number` and handling `undefined` states with `??`).
